# 기획 체크리스트

## 카테고리 1: 게임 컨셉 (필수)

- [ ] 게임 한 줄 설명 (엘리베이터 피치)
- [ ] 장르
- [ ] 카메라 시점 (1인칭/3인칭/탑다운/사이드뷰)
- [ ] 타겟 플랫폼 (PC/모바일/콘솔/크로스플랫폼)
- [ ] 핵심 경험 (플레이어가 느낄 감정)
- [ ] 레퍼런스 게임 및 차별점

---

## 카테고리 2: 게임플레이 (필수)

- [ ] 코어 루프
- [ ] 핵심 메카닉 (3-5개)
- [ ] 승리/패배 조건
- [ ] 기본 수치 (속도, 체력, 쿨다운 등)

---

## 카테고리 3: 콘텐츠 (필수)

- [ ] 캐릭터/역할 정의
- [ ] 레벨/맵 설계 개요
- [ ] UI 플로우 (생략 가능)

---

## 카테고리 4: 멀티플레이어/네트워크 (해당 시)

**싱글플레이어 전용이면 스킵**

- [ ] 연결 방식 (P2P 호스트/데디케이티드 서버/클라우드)
- [ ] 최대 플레이어 수
- [ ] 매칭 방식 (호스트 초대/매치메이킹/로비)
- [ ] 호스트 이탈 처리
- [ ] 레이턴시 보상 방식
- [ ] 동기화 대상

---

## 카테고리 5: 크로스플랫폼/모바일 (해당 시)

**PC 전용이면 스킵**

- [ ] 터치 UI 배치 (조이스틱 위치, 버튼 크기/위치)
- [ ] 화면 비율 대응 (16:9, 18:9, 노치 등)
- [ ] 모바일 성능 목표 (목표 FPS, 최소 사양)
- [ ] 배터리/발열 고려사항
- [ ] PC-모바일 밸런스 차이

---

## 카테고리 6: 사운드 & 피드백 (선택)

- [ ] BGM 스타일/분위기
- [ ] 주요 효과음 (액션, UI, 환경)
- [ ] 햅틱/진동 피드백
- [ ] 시각적 피드백 (화면 효과, 파티클)

---

## 카테고리 7: 온보딩 & 튜토리얼 (선택)

- [ ] 튜토리얼 방식
- [ ] 첫 플레이 경험 흐름
- [ ] 도움말/힌트 시스템

---

## 카테고리 8: 설정 & 접근성 (선택)

- [ ] 그래픽 설정 옵션
- [ ] 사운드 설정
- [ ] 조작 설정
- [ ] 접근성 옵션

---

## 카테고리 9: 메타 시스템 (선택)

- [ ] 통계/전적 기록
- [ ] 업적/도전과제
- [ ] 보상/언락 시스템
- [ ] 로컬라이제이션

---

## 카테고리 10: 기술 (필수)

- [ ] Unity 버전 & 렌더 파이프라인
- [ ] 필수 패키지
- [ ] 폴더 구조
- [ ] 네이밍 컨벤션

---

## 진행 가이드

1. **필수 카테고리**(1-3, 10)는 반드시 확인
2. **조건부 카테고리**(4-5)는 해당 여부 먼저 확인
3. **선택 카테고리**(6-9)는 간단히 물어보고 "나중에"면 스킵
4. 2-3개씩 끊어서 질문
