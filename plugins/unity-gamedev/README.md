# unity-gamedev

유니티 게임 개발: 맵 빌더, 게임 기획, 프로토타입

## Skills

| Skill | Description |
|-------|-------------|
| unity-map-builder | ProBuilder 기반 맵/레벨 자동 생성 |
| unity-planning | 게임 기획 문서 작성 (GDD) |
| unity-prototype | 기획 → 프로토타입 구현 |
| unity-task-decomposer | 기획 문서 → MCP 태스크 분해 |

### 워크플로우

```
unity-planning → unity-task-decomposer → unity-prototype
     ↓
unity-map-builder (맵 생성)
```
