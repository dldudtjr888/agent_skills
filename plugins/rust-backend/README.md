# rust-backend

## Skills

| Skill | Description |
|-------|-------------|
| rust-patterns | Rust 개발 패턴 및 베스트 프랙티스 |

### rust-patterns 주요 내용

- Ownership/Borrowing 패턴
- Error Handling (thiserror vs anyhow)
- Trait 설계 원칙
- 동시성 (Tokio, Message Passing)
- Builder/Typestate 패턴
