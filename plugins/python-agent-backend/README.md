| Skill | Description |
|-------|-------------|
| pydantic-ai-builder | Pydantic AI 에이전트 빌더 |
| langgraph-builder | LangGraph 워크플로우 빌더 |
| agents-sdk-builder | OpenAI Agents SDK 빌더 |

| langchain-openrouter | LangChain + OpenRouter |

| google-adk-builder | Google ADK 에이전트 빌더 |
